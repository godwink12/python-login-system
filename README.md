# Simple Python Login System
A terminal-based authentication system built with Python.

## Features
- User Registration (Username & Password)
- Secure Login validation
- Basic error handling for incorrect credentials

## How to Run
1. Ensure you have Python installed.
2. Run `python main.py` in your terminal.
3. Follow the on-screen prompts to register or login.

## Future Improvements
- Implement password hashing for security.
- Connect to a SQLite database.
